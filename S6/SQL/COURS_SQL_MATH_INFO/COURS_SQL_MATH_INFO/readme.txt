1) Les répertoires

Le répertoire cours
	Ce répertoire contient les supports suivants :
	- MCD MLD Merise
	- Cours SQL Oracle 
	- Cours PLSQL et procédures stockées
	- Cours BD relationnelles (pour complément d'information).
	
	Voir aussi le fichier readme local à ce répertoire.

Le répertoire envEtConnexion
	Contient les informations sur les différentes possibles de se connecter
	Via SQLPLUS, Via SQLDeveloper ainsi que les procédures d'installation du SGBD Oracle.
	

Le répertoire Exercices
	Contient les fichiers des énoncés des exercices des cours MCD / MLD MERISE,  
	Procédures stockées. Les exercices du cours SQL et PLSQL sont intercallés dans
	le support de cours correspondant.

Le répertoire scripts
	contient les différents scripts nécessaires pour les TP.
	Voir le fichier readme dans ce répertoire pour plus de détails.

Le répertoire TP transversal
	Contient le TP d'approfondissement à réaliser par groupe d'étudiants

Les supports seront disponibles sur L'ENT.
